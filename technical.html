CSS (Cascading Style Sheets) allows you to create great-looking web pages. This article explains what CSS is, its basic syntax, and how your browser applies CSS to HTML.

Browser Default Styles

Browsers apply basic styles to HTML elements by default. For example, headings are larger, paragraphs are separated by space, and links are underlined and colored.

These styles come from the browser's default stylesheet, ensuring web pages are readable even without explicit styling.

What is CSS For

CSS controls the presentation of HTML documents. It allows you to customize colors, fonts, layouts, and animations.

  • Text styling: Change font size, color, and weight.
  • Layouts: Create multi-column layouts.
  • Special effects: Add animations and transitions.

CSS Syntax Basics

CSS is a rule-based language where you define rules to style HTML elements.

Example of a CSS rule:

h1 { 
    color: red; 
    font-size: 2.5em; 
}

Explanation:

  • The selector (h1) targets elements.
  • Curly braces {} contain style declarations.
  • Each declaration has a property (color) and a value (red).

Example of a CSS rule for a paragraph:

p {
    color: green;
    font-size: 16px;
}

Another CSS example to set the background color:

body {
    background-color: #fafafa;
}

How is CSS Applied to HTML

When a browser loads a web page, it follows these steps:

  1. Parses the HTML document.
  2. Loads and applies CSS styles.
  3. Builds a render tree and displays the page.

Example:

<h1>CSS is great</h1>
<p>You can style text and layouts.</p>
button {
    padding: 10px;
    border-radius: 5px;
    background-color: orange;
}
